# 數字易經分析

這是一個 React + Vite 製作的數字易經分析工具。

## 本機執行

```bash
npm install
npm run dev
```

開啟終端機顯示的本機網址即可使用。

## 建置

```bash
npm run build
```

完成後會產生 `dist` 資料夾。

## 上傳到 GitHub

1. 在 GitHub 建立新的 repository。
2. 將此資料夾內的檔案上傳到 repository。
3. 若要部署到 GitHub Pages，可使用 Vite 的 GitHub Pages 部署流程，或把 `dist` 內容部署到 Pages。
