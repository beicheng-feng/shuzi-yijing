# 數字易經

這是一個 React 版的數字易經分析工具，已整理成可以上傳到 GitHub 的 Vite 專案。

## 本機預覽

```bash
npm install
npm run dev
```

## 建置正式版

```bash
npm run build
```

建置後的檔案會輸出在 `dist` 資料夾。

## 發布到 GitHub Pages

1. 將整個資料夾上傳或推送到 GitHub repository。
2. 到 repository 的 `Settings` → `Pages`。
3. 在 `Build and deployment` 選擇 `GitHub Actions`。
4. 推送到 `main` 分支後，GitHub 會自動建置並發布網站。
